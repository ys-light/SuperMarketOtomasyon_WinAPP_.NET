**Proje Hakkında**

SuperMarket Automation, C# Windows Forms kullanılarak geliştirilmiş masaüstü tabanlı bir market otomasyon uygulamasıdır. Proje; market yönetimini kolaylaştırmak amacıyla ürün, kullanıcı ve satış işlemlerinin tek bir uygulama üzerinden yönetilmesini sağlamaktadır.

Uygulama içerisinde farklı kullanıcı panelleri bulunmakta olup, yönetici ve kasiyer gibi kullanıcı rollerine göre farklı işlemler gerçekleştirilebilmektedir.

**Kullanılan Teknolojiler**
-C#
-.NET Framework
-Windows Forms (WinForms)
-Microsoft SQL Server
-ADO.NET
-Visual Studio
 
 **Amaç**
Bu proje, masaüstü uygulama geliştirme ve SQL Server ile veritabanı işlemlerini uygulamalı olarak öğrenmek amacıyla geliştirilmiştir. Kullanıcı yönetimi, ürün işlemleri ve satış süreçlerini tek bir sistem üzerinden yöneterek gerçek bir market otomasyonunun temel yapısını oluşturmayı hedeflemektedir.

**Temel Özellikler**
- Kullanıcı giriş sistemi
- Yönetici ve kasiyer panelleri
- Ürün yönetimi
- Meyve & sebze kategorisi işlemleri
- SQL Server veritabanı entegrasyonu
- Veri ekleme, güncelleme, silme ve listeleme (CRUD)
